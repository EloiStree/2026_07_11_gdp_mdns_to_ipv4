# 2026_07_11_gdp_mdns_to_ipv4

Try to turn a mDNS to IPV4 in Godot.  

I am using Raspberry Pi on local network.    
Sometime other device that can generate mDNS.    
You can also use no-ip.   

The idea here is to provide some code that allows to conver this address to IPV4 and check if the address is reachable.
